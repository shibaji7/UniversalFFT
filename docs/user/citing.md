# Citing UniversalFFT

If you use UniversalFFT in your research, please cite the software and the
foundational paper that defines the conventions it implements.

---

## Citing the software

```bibtex
@software{Chakraborty_UniversalFFT_2026,
  author    = {Chakraborty, Shibaji},
  title     = {{UniversalFFT}: A multi-language {FFT/IFFT} wrapper library
               for physically correct geoscience {F}ourier transforms},
  year      = {2026},
  version   = {1.0.0},
  doi       = {10.5281/zenodo.XXXXXXX},
  url       = {https://github.com/shibaji7/UniversalFFT}
}
```

---

## Citing the foundational paper

All conventions implemented in this library are described in:

```bibtex
@article{Boteler2012,
  author  = {Boteler, David H.},
  title   = {On Choosing {F}ourier Transforms for Practical Geoscience Applications},
  journal = {International Journal of Geosciences},
  year    = {2012},
  volume  = {3},
  pages   = {952--959},
  doi     = {10.4236/ijg.2012.325096},
  url     = {https://doi.org/10.4236/ijg.2012.325096}
}
```
