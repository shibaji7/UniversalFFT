## Contributing

Contributions are welcome. Please open an Issue or Pull Request on the
[UniversalFFT GitHub repository](https://github.com/shibaji7/UniversalFFT).

When adding a new language implementation, please ensure:

1. The implementation passes the cross-language validation harness
   (`tests/cross_language/validate_all.py`) within a tolerance of \(10^{-9}\).
2. A corresponding demo script (analogous to `c/demo.c`, `matlab/ufft_demo.m`,
   `r/ufft_demo.R`) is provided.
3. API documentation is added under `docs/dev/<language>/`.
